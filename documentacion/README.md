# Documentación del proceso completo
# Este proyecto contiene scripts para la gestión de infraestructura de servidores remotos.

## Configuración Inicial
El script `setup_server.sh` realiza la configuración inicial del servidor remoto.

## Monitoreo de Recursos
El script `monitor_resources.sh` permite el monitoreo de recursos del servidor.

## Aplicación de Parches
El script `apply_patches.sh` automatiza la aplicación de parches de seguridad en el servidor.

## Requisitos
- Acceso SSH al servidor remoto
- Scripts de configuración y parches preparados

## Uso
1. Edita las variables de entorno en los scripts según tu configuración.
2. Ejecuta los scripts en orden: configuración, monitoreo y parches.
